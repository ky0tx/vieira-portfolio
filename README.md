# Vieira — Portfólio de Design Gráfico

Portfólio pessoal criado para apresentar meu trabalho e serviços de design gráfico.

## Sobre

Este projeto reúne uma seleção dos meus trabalhos de design gráfico, com o objetivo de apresentar meu estilo, processo e serviços para clientes em potencial.

## Tecnologias

- HTML
- JavaScript
- Desenvolvido com o auxílio do Claude (Anthropic)

```
vieira/

```

> Ajuste essa estrutura conforme os arquivos reais do seu projeto.

## Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/vieira.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd vieira
   ```
3. Abra o arquivo `index.html` no navegador, ou use uma extensão como Live Server.
